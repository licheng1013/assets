## Navicat16激活
- 1: 需要管理员选择里面的激活权限 > 勾选hosts >  选择安装目录 > 生成激活码 > 
- 2: 打开navicat16 > 输入激活码 > 选择手动激活 > 复制激活码到激活码软件 > 生成另一个激活码 > 复制到navicat激活完成
